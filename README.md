# Aqua Connect Exim

## Project Overview
Aqua Connect Exim is a web application designed for an Indian spices importer and exporter that connects farmers to global markets. The platform showcases a variety of high-quality spices, processed foods, and tea products while providing information about their services, certifications, and global reach. 

## Installation
To run this project locally, follow the steps below:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/aqua-connect-exim.git
   cd aqua-connect-exim
   ```

2. Open the `index.html` file in your preferred web browser.

## Usage
Once you have everything set up, simply open the `index.html` file in a browser. You will be able to explore the different sections of the website, including:

- Home
- About
- Products
- Services
- Testimonials
- FAQs
- Contact Information

The homepage provides buttons to inquire about products and download brochures.

## Features
- **Responsive Design:** The website layout adapts seamlessly to different screen sizes.
- **Product Showcase:** Detailed presentation of various spices and food products.
- **User Engagement:** Modal pop-ups for inquiries and brochure downloads.
- **Testimonials Section:** Displaying feedback from clients.
- **Certifications Information:** Showcasing global certifications for quality assurance.
- **Multilingual Support:** Languages available for a broader audience.

## Dependencies
The project includes a CSS stylesheet referenced in the `index.html` file. Below are the resources utilized:

- Google Fonts for typography.
- Custom CSS for styles located in `assets/css/styles.css`.
- JavaScript functionalities for modal operations included in `assets/js/scripts.js`.

No additional dependencies are specified in a `package.json` since this project does not use Node.js or any server-side dependencies.
